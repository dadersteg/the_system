| Agent | Role | When to Summon (Triggers) | The "Magic Phrase" |
| --- | --- | --- | --- |
| **System Architect** | The Governor (Legislative) | • You need a new Project Code.<br>• You have a complex item that fits nowhere.<br>• You have messy notes to turn into a Protocol. | "Where does this new project fit in the LOS?"<br>"Here are some ramblings, turn this into a Protocol." |
| **Taskmaster** | The Doer (Executive) | • You need to add/check tasks.<br>• You want to run your Daily/Weekly Review.<br>• You have a Report to process. | "Start daily review."<br>"Remember this: [Fact]." |

| **Atlas** | The Analyst (Reflective) | • You want to run a Daily Reflection.<br>• You want to synthesize qualitative journals with Vantage data.<br>• You need strategic realignment for goals. | "Start daily reflection."<br>"Review my 90-day progress." |

### 🔄 THE WORKFLOW LOOP

**1. The "New Project" Scenario**
* Step 1 (Architect): "I'm starting a project on 'Coffee Brewing'. Give me a code." -> Result: Creates `01 05 01 Projects > Coffee`.
* Step 2 (Taskmaster): "Create a task to buy beans for the Coffee project." -> Result: Adds task to Google Tasks.
* Step 3 (Clerk): (Upload Receipt) -> Result: Names file `Coffee - Receipt (Beans).pdf` and gives path `01 05 01 Projects > Coffee`.

**2. The "Reflection & Audit" Cycle**
* Step 1 (Vantage): (Upload Gym log) -> Result: Outputs Clinical Performance Audit Table mapped to URNs.
* Step 2 (Atlas): (Upload Vantage Audit + Voice Note) -> Result: Connects the raw data to personal feelings and recommends habit changes.
* Step 3 (Taskmaster): "Execute Atlas's new habit recommendation." -> Result: Creates new recurring task.

### ⚠️ CRITICAL RULES (THE "GOTCHAS")

1. Don't ask Taskmaster to file. It can't see your folders. It will hallucinate paths. Use The Clerk.
2. Don't ask The Clerk to invent codes. It is strict. If a code doesn't exist, it will use the Parent L3 or ask for help. Use System Architect.
3. Don't ask Architect to "do" things. It creates the plan; it doesn't execute the task. Use Taskmaster.
4. Don't ask Vantage for advice. Vantage is purely clinical data extraction. Use Atlas for strategic synthesis and advice.
