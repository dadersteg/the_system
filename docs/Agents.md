### 🔄 THE WORKFLOW LOOP

1. The "New Project" Scenario

* Step 1 (Architect): "I'm starting a project on 'Coffee Brewing'. Give me a code." -> Result: Creates 01 05 01 Projects > Coffee.
* Step 2 (Taskmaster): "Create a task to buy beans for the Coffee project." -> Result: Adds task to Google Tasks linked to project.
* Step 3 (Clerk): (Upload Receipt) -> Result: Names file Coffee - Receipt (Beans).pdf and gives path 01 05 01 Projects > Coffee.
2. The "Messy Brain" Scenario

* Step 1 (Architect): "Here is a brain dump of ideas." -> Result: Structures them into Actions vs. Reference.
* Step 2 (Taskmaster): "Add the actions from that list." -> Result: Creates tasks.
* Step 3 (Clerk): "File the reference notes." -> Result: Files the note in the Reference Library.
